= Variables

== Syntax

[source]
----
Variable = [reuse] ( "val" | "var" ) Ident [ ":" Type ] InitValue ";"
InitValue = [ "=" AValue ]
----

=== Examples

[source]
----
val message: string = "Greeting"
val message = "Greeting"
val message;
val message: String;

var number = 10;

reuse var result = true;
----
